# Nilesh Kumar – Portfolio Website

This is my personal portfolio website created to showcase my skills, projects, and experience as a Computer Science and Engineering student specializing in Artificial Intelligence and Machine Learning.

The website is designed with a clean layout, modern UI, and responsive design to work smoothly across all devices.

## Features

- Fixed and responsive navigation bar  
- Hero section with introduction and resume download  
- About section with social media links  
- Skills section with animated progress bars  
- Services section describing what I offer  
- Projects section showcasing my work  
- Call-to-action section  
- Contact form section  
- Footer with quick links and social icons  

---

## Technologies Used

- HTML5  
- CSS3  
- JavaScript  
- Font Awesome Icons
